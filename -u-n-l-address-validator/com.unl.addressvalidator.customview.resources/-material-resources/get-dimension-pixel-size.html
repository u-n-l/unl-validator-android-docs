getDimensionPixelSize

open fun getDimensionPixelSize(@NonNull() context: Context, @NonNull() attributes: TypedArray, @StyleableRes() index: Int, defaultValue: Int): Int
Content copied to clipboard

Retrieve a dimensional unit attribute at index for use as a size in raw pixels. Asize conversion involves rounding the base value, and ensuring that a non-zero base value is atleast one pixel in size.

This method will throw an exception if the attribute is defined but is not a dimension.

Return

Attribute dimension value multiplied by the appropriate metric and truncated to integerpixels, or defaultValue if not defined.
